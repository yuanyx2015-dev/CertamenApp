## Script Notes

**Total Runtime:** ~4:35 minutes

**Tone:** Energetic, educational, inspiring—appeals to both competitive students and casual learners

**Key Messages:**
1. Easy to start (simple authentication)
2. Comprehensive practice with adaptive difficulty
3. Track progress with ranks and stats
4. Review and improve with targeted study tools
5. Exciting multiplayer features coming soon
6. Professional, reliable technology

**Visual Style Suggestions:**
- Heavy use of Roman-themed graphics (laurel wreaths, meander borders, Roman colors)
- Screen recordings showing actual app gameplay
- Animated transitions between sections
- Highlight UI elements with circles/arrows when explaining features
- Show rank progression with animated badge transitions
- Use quick cuts during feature montages to maintain energy

**Music Suggestions:**
- Epic/cinematic orchestral background track with Roman/Latin feel
- Subtle sound effects for buzz-ins and correct answers
- Volume dip during narration for clarity

**Call-to-Action Options:**
- End screen with download links (App Store, Google Play)
- Social media handles
- Website URL
- "Join the community" message
